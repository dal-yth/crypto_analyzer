Crypto Analyzer API

downward_trend

GET /downward_trend Get longest downward trend from range of dates.
Parameters  
   
Name Description
coin id of coin (bitcoin, ethereum, etc.), default: bitcoin
vs_currency The target currency of market data (usd, eur, jpy, etc.), default: eur
from* From date in format: YYYY-MM-DD (eg. 2021-01-01)
to* To date in format: YYYY-MM-DD (eg. 2021-02-01)
   
*required field  
Responses  
   
 Code Description
 200 Get longest downward trend of historical market data.
